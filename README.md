## アプリ概要

自分の単語帳を複数作り管理できるアプリです。<br>

- 複数の単語帳を作成することができます。
- 単語帳を指定し学習することができます。
- 単語帳作成後であっても新しい単語の追加、消去をすることができます。<br><br>

## 使用した技術

- フロントエンド

  - React
  - Redux
  - material-UI

- サーバーサイド、デプロイ

  - Firebase
    - Authentication (ユーザー認証)
    - Firestores (データベース)
    - Cloud functions (関数の実行)
